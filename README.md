ccn-mobility
============

Simple scenario and files for mobility in ndnSIM

To use this scenario template, a complete install of ns-3 and ndnSIM is
required. Please consult the respective documentation.

Ns-3
http://www.nsnam.org/

ndnSIM
http://ndnsim.net/index.html
